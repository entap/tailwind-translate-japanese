## 使用方法

`bg-clip-{keyword}`ユーティリティを使用して、要素の背景の境界ボックスを制御します。

```html
<template preview>
  <div class="flex justify-around">
    <div class="flex-1">
      <p class="text-center text-sm text-gray-600 mb-1">.bg-clip-border</p>
      <div class="mx-auto bg-purple-500 w-24 h-24 bg-clip-border border-4 border-dashed border-purple-800 p-4"></div>
    </div>
    <div class="flex-1">
      <p class="text-center text-sm text-gray-600 mb-1">.bg-clip-padding</p>
      <div class="mx-auto bg-purple-500 w-24 h-24 bg-clip-padding border-4 border-dashed border-purple-800 p-4"></div>
    </div>
    <div class="flex-1">
      <p class="text-center text-sm text-gray-600 mb-1">.bg-clip-content</p>
      <div class="mx-auto bg-purple-500 w-24 h-24 bg-clip-content border-4 border-dashed border-purple-800 p-4"></div>
    </div>
  </div>
</template>

<div class="bg-clip-border p-4 border-4 border-dashed border-purple-800 bg-purple-500"></div>
<div class="bg-clip-padding p-4 border-4 border-dashed border-purple-800 bg-purple-500"></div>
<div class="bg-clip-content p-4 border-4 border-dashed border-purple-800 bg-purple-500"></div>
```

## テキストのトリミング

`bg-clip-text`を使用して、テキストの形状に一致するように要素の背景をトリミングします。 テキストを通して背景画像を表示したい効果に役立ちます。

```html
<template preview class="p-8">
  <div class="text-center text-5xl font-extrabold leading-none tracking-tight">
    <span class="bg-clip-text text-transparent bg-gradient-to-r from-teal-400 to-blue-500">
      Hello world
    </span>
  </div>
</template>
```

## レスポンシブ

特定のブレークポイントで要素の背景の境界ボックスを制御するには、既存の背景クリップユーティリティに `{screen}:`プレフィックスを追加します。 たとえば、クラス`md:bg-clip-padding`を要素に追加すると、中程度の画面サイズ以上で`bg-clip-padding`ユーティリティが適用されます。

```html
<div class="bg-clip-padding md:bg-clip-border">
  <!-- ... -->
</div>
```

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。
