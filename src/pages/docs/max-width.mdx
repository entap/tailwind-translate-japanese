## 使用方法

`max-w-{size}`ユーティリティを使用して、要素の最大幅を設定します。

```html
<template preview>
  <div class="max-w-md mx-auto text-center p-6 bg-gray-300">
    max-w-md
  </div>
</template>

<div class="max-w-md mx-auto ...">
  max-w-md
</div>
```

## レスポンシブ

特定のブレークポイントで要素の最大幅を制御するには、既存の最大幅ユーティリティに`{screen}:`プレフィックスを追加します。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['max-w-sm', 'max-w-md', 'max-w-lg', 'max-w-xl', 'max-w-2xl']}
  snippet={(classNames) => `
    <div class="${classNames} ...">
      Target
    </div>
  `}
  preview={(className) => (
    <div className={`mx-auto text-center p-6 bg-gray-300 ${className}`}>
      Target
    </div>
  )}
/>

## カスタマイズ

### Max-Width Scale

`tailwind.config.js`ファイルの` theme.maxWidth`セクションでTailwindのデフォルトの最大幅スケールをカスタマイズします。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      maxWidth: {
+       '1/4': '25%',
+       '1/2': '50%',
+       '3/4': '75%',
      }
    }
  }
```

デフォルトテーマのカスタマイズについては[theme customization documentation](/docs/theme#customizing-the-default-theme)をご覧ください。
