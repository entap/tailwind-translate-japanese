## Baseline

`.align-baseline`を使用して、要素のベースラインをその親のベースラインに揃えます。

```html
<template preview>
  <div class="leading-none relative">
    <span class="w-1 h-8 inline-block align-baseline bg-blue-300">
      <span class="absolute top-0 border-blue-300 border-t border-b w-full h-8"></span>
      <span class="absolute top-0 border-blue-300 border-t border-b w-full h-4"></span>
    </span>
    <span class="relative z-10">.align-baseline</span>
  </div>
</template>

<span class="inline-block align-baseline ...">...</span>
```

## Responsive

特定のブレークポイントでのみ垂直方向の配置を制御するには、既存の垂直方向の配置ユーティリティに`{screen}:`プレフィックスを追加します。
たとえば、クラス`md:align-top`を要素に追加すると、中程度の画面サイズ以上で`align-top`ユーティリティが適用されます。

Tailwindのレスポンシブデザイン機能の詳細については、[レスポンシブデザイン](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['align-top', 'align-middle', 'align-bottom', 'align-text-top', 'align-text-bottom']}
  snippet={(classNames) => `
    <div class="leading-none relative">
      <span class="${classNames} ...">...</span>
      <span>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</span>
    </div>
  `}
  preview={(className) => (
    <div className="leading-none relative">
      <span className={`w-1 h-8 inline-block bg-blue-300 ${className}`}>
        <span className="absolute top-0 border-blue-300 border-t border-b w-full h-8" />
        <span className="absolute top-0 border-blue-300 border-t border-b w-full h-4" />
      </span>
      <span className="relative z-10">Lorem ipsum dolor sit amet, consectetur adipisicing elit.</span>
    </div>
  )}
/>
