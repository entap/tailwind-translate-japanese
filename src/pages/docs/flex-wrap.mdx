## Don't wrap

`.flex-no-wrap`を使用して、フレックスアイテムがラップされないようにし、必要に応じて柔軟性のないアイテムがコンテナからオーバーフローするのを防ぎます。

```html
<template preview>
  <div class="flex flex-no-wrap bg-gray-200">
    <div class="w-2/5 flex-none p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">1</div>
    </div>
    <div class="w-2/5 flex-none p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">2</div>
    </div>
    <div class="w-2/5 flex-none p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">3</div>
    </div>
  </div>
</template>
```

## Wrap normally

`.flex-wrap`を使用して、フレックスアイテムをラップできるようにします。

```html
<template preview>
  <div class="flex flex-wrap bg-gray-200">
    <div class="w-2/5 p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">1</div>
    </div>
    <div class="w-2/5 p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">2</div>
    </div>
    <div class="w-2/5 p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">3</div>
    </div>
  </div>
</template>
```

## Wrap reversed

`.flex-wrap-reverse`を使用して、フレックスアイテムを逆方向にラップします。

```html
<template preview>
  <div class="flex flex-wrap-reverse bg-gray-200">
    <div class="w-2/5 p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">1</div>
    </div>
    <div class="w-2/5 p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">2</div>
    </div>
    <div class="w-2/5 p-2">
      <div class="text-gray-700 text-center bg-gray-400 p-2">3</div>
    </div>
  </div>
</template>
```

## Responsive

フレックスアイテムが特定のブレークポイントでどのようにラップするかを制御するには、既存のユーティリティクラスに`{screen}:`プレフィックスを追加します。
たとえば、`md:flex-wrap-reverse`を使用して、中程度の画面サイズ以上でのみ`flex-wrap-reverse`ユーティリティを適用します。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['flex-no-wrap', 'flex-wrap', 'flex-wrap-reverse', 'flex-no-wrap', 'flex-wrap']}
  snippet={(classNames) => `
    <div class="${classNames} ...">
      <!-- ... -->
    </div>
  `}
  preview={(className) => (
    <div className={`flex bg-gray-200 ${className}`}>
      <div className="w-2/5 flex-none p-2">
        <div className="text-gray-700 text-center bg-gray-400 p-2">1</div>
      </div>
      <div className="w-2/5 flex-none p-2">
        <div className="text-gray-700 text-center bg-gray-400 p-2">2</div>
      </div>
      <div className="w-2/5 flex-none p-2">
        <div className="text-gray-700 text-center bg-gray-400 p-2">3</div>
      </div>
    </div>
  )}
/>
