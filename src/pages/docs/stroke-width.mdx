## 使用方法

`.stroke-{width}`ユーティリティを使用して、SVGのストローク幅を設定します。

完全にストロークで描画される[Feather](https://feathericons.com/) のようなアイコンセットのスタイリングに役立ちます。


## Responsive

特定のブレークポイントでSVG要素のストローク幅を制御するには、既存の幅ユーティリティに`{screen}:`プレフィックスを追加します。
たとえば、`md:stroke-2`クラスを要素に追加すると、中程度の画面サイズ以上で`stroke-2`ユーティリティが適用されます。

Tailwindのレスポンシブデザイン機能の詳細については、[レスポンシブデザイン](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['stroke-1', 'stroke-2', 'stroke-1', 'stroke-0', 'stroke-1']}
  snippet={(classNames) => `

    </svg>
  `}
  previewClassName="text-center p-4"
  preview={(className) => (
    <svg className={`stroke-current text-green-500 inline-block h-24 w-24 ${className}`} viewBox="0 0 24 24" fill="none" strokeLinecap="round" strokeLinejoin="round">
      <circle cx="12" cy="12" r="10" />
      <line x1="14.31" y1="8" x2="20.05" y2="17.94" />

## Customizing

`tailwind.config.js`ファイルの`theme.strokeWidth`セクションをカスタマイズして、Tailwindが生成するストローク幅ユーティリティを制御します。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        strokeWidth: {
+         '3': '3',
+         '4': '4',
        }
      }
    }
  }
```

デフォルトテーマのカスタマイズについては[テーマのカスタマイズ](/docs/theme#customizing-the-default-theme)をご覧ください。
