## 使用方法

`min-h-0`,`min-h-full`または`min-h-screen`ユーティリティを使用して、要素の最小の高さを設定します。

```html
<template preview>
  <div class="h-48 p-6 bg-gray-300">
    <div class="h-24 min-h-full p-6 bg-gray-400 flex items-center justify-center">
      <span>min-h-full</span>
    </div>
  </div>
</template>

<div class="h-48 ...">
  <div class="h-24 min-h-full ...">
    min-h-full
  </div>
</div>
```

## レスポンシブ

特定のブレークポイントで要素の最小の高さを制御するには、既存の最小の高さユーティリティに`{screen}:`プレフィックスを追加します。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['min-h-full', 'min-h-0', 'min-h-full', 'min-h-0', 'min-h-full']}
  snippet={(classNames) => `
    <div class="h-48 ...">
      <div class="h-24 ${classNames} ...">
        Target
      </div>
    </div>
  `}
  preview={(className) => (
    <div className="h-48 p-6 bg-gray-300">
      <div className={`h-24 p-6 bg-gray-400 flex items-center justify-center ${className}`}>
        <span>Target</span>
      </div>
    </div>
  )}
/>

## カスタマイズ

### Min-height scale

`tailwind.config.js`ファイルの`theme.minHeight`セクションでTailwindのデフォルトの最小高さスケールをカスタマイズします。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      minHeight: {
+       '0': '0',
+       '1/4': '25%',
+       '1/2': '50%',
+       '3/4': '75%',
+       'full': '100%',
      }
    }
  }
```

デフォルトテーマのカスタマイズについては[theme customization documentation](/docs/theme#customizing-the-default-theme)をご覧ください。
