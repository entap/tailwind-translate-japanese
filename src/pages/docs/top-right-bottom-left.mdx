## 使用方法

`.{top|right|bottom|left|inset}-0`ユーティリティを使用して、絶対位置にある要素を最も近い位置にある親の端のいずれかに固定します。

Tailwindのパディングおよびマージンユーティリティと組み合わせると、絶対位置の要素を正確に制御するために必要なのはこれらだけであることがわかるでしょう。

## Responsive

特定のブレークポイントにのみ要素を配置するには、既存の配置ユーティリティに`{screen}:`プレフィックスを追加します。
たとえば、クラス`md:inset-y-0`を要素に追加すると、中程度の画面サイズ以上で`inset-y-0`ユーティリティが適用されます。

Tailwindのレスポンシブデザイン機能の詳細については、[レスポンシブデザイン](/docs/responsive-design)のドキュメントをご覧ください。

    </div>

      <div className={`absolute bg-gray-800 p-4 text-gray-500 ${className}`}>Responsive element</div>
    </div>

/>

## Customizing

### Top / Right / Bottom / Left scale

デフォルトでは、Tailwindは`0`および`auto`のtop/right/bottom/left/insetユーティリティのみを提供します。
`tailwind.config.js`ファイルの`theme.inset`セクションを編集することで、これらを変更、追加、または削除できます。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      inset: {
        '0': 0,
-       auto: 'auto',
+       '1/2': '50%',
      }
    }
  }
```

### Negative values

Tailwindの[negative margin](/docs/margin＃negative-margins)クラスと同じ形式の負のtop/right/bottom/leftクラスを追加する場合は、構成ファイルのキーの前にダッシュを付けます。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        inset: {
+         '-16': '-4rem',
        }
      }
    }
  }
```

Tailwindは、予想されるような`top--16`ではなく、先頭のダッシュが表示されたときに`-top-16`のようなクラスを生成するのに十分スマートです。
