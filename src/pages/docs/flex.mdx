## None

`.flex-none`を使用して、フレックスアイテムが拡大または縮小するのを防ぎます。

```html
<template preview>
  <div class="flex bg-gray-200">
    <div class="flex-1 text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">
      Item that can grow or shrink if needed
    </div>
    <div class="flex-none text-gray-800 text-center bg-gray-500 px-4 py-2 m-2">
      Item that cannot grow or shrink
    </div>
    <div class="flex-1 text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">
      Item that can grow or shrink if needed
    </div>
  </div>
</template>
```

## Responsive

フレックスアイテムが特定のブレークポイントでどのように拡大および縮小するかを制御するには、既存のユーティリティクラスに `{screen}:`プレフィックスを追加します。
たとえば、`md:flex-1`を使用して、中程度の画面サイズ以上でのみ`flex-1`ユーティリティを適用します。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['flex-none', 'flex-1', 'flex-auto', 'flex-initial', 'flex-1']}
  snippet={(classNames) => `
    <div class="flex ...">
      <!-- ... -->
      <div class="${classNames} ...">
        Responsive flex item
      </div>
      <!-- ... -->
    </div>
  `}
  preview={(className) => (
    <div className="flex bg-gray-200">
      <div className="flex-1 text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">
        Item that can grow or shrink if needed
      </div>
      <div className={`text-gray-800 text-center bg-gray-500 px-4 py-2 m-2 ${className}`}>
        Responsive flex item
      </div>
      <div className="flex-1 text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">
        Item that can grow or shrink if needed
      </div>
    </div>
  )}
/>

## カスタマイズ

### Flex Values

デフォルトでは、Tailwindは4つの`flex`ユーティリティを提供します。
Tailwind構成の`theme.flex`セクションを編集して、これらを変更、追加、または削除します。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      flex: {
        '1': '1 1 0%',
        auto: '1 1 auto',
-       initial: '0 1 auto',
+       inherit: 'inherit',
        none: 'none',
+       '2': '2 2 0%',
      }
    }
  }
```
