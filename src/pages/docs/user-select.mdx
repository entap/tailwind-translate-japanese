## Disable selecting text

要素とその子のテキストを選択しないようにするには、`.select-none`を使用します。

```html
<template preview class="text-center p-4">
  <div class="mx-auto max-w-sm bg-gray-200 p-2 select-none">
    This text is not selectable
  </div>
</template>

<div class="select-none ...">
  This text is not selectable
</div>
```

## Allow selecting text

`.select-text`を使用して、要素とその子のテキストを選択できるようにします。

```html
<template preview class="text-center p-4">
  <div class="mx-auto max-w-sm bg-gray-200 p-2 select-text">
    This text is selectable
  </div>
</template>

<div class="select-text ...">
  This text is selectable
</div>
```

## Select all text in one click

ユーザーがクリックしたときに要素内のすべてのテキストを自動的に選択するには、`.select-all`を使用します。

```html
<template preview class="text-center p-4">
  <div class="mx-auto max-w-sm bg-gray-200 p-2 select-all">
    Click anywhere in this text to select it all
  </div>
</template>

<div class="select-all ...">
  Click anywhere in this text to select it all
</div>
```

## Auto

テキストの選択にデフォルトのブラウザ動作を使用するには、`.select-auto`を使用します。異なるブレークポイントで`.select-none`のような他のクラスを元に戻すのに便利です。

```html
<template preview class="text-center p-4">
  <div class="mx-auto max-w-sm bg-gray-200 p-2 select-auto">
    This text is selectable
  </div>
</template>

<div class="select-auto ...">
  This text is selectable
</div>
```
