## Row

`.flex-row`を使用して、フレックスアイテムをテキストと同じ方向に水平に配置します。

```html
<template preview>
  <div class="flex flex-row bg-gray-200">
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">1</div>
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">2</div>
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">3</div>
  </div>
</template>
```

## Row reversed

`.flex-row-reverse`を使用して、フレックスアイテムを反対方向に水平に配置します。

```html
<template preview>
  <div class="flex flex-row-reverse bg-gray-200">
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">1</div>
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">2</div>
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">3</div>
  </div>
</template>
```

## Column

`.flex-col`を使用して、フレックスアイテムを垂直に配置します。

```html
<template preview>
  <div class="flex flex-col bg-gray-200">
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">1</div>
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">2</div>
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">3</div>
  </div>
</template>
```

## Column reversed

`.flex-col-reverse`を使用して、フレックスアイテムを反対方向に垂直に配置します。

```html
<template preview>
  <div class="flex flex-col-reverse bg-gray-200">
    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">1</div>

    <div class="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">3</div>
  </div>
</template>
```

## Responsive

特定のブレークポイントでのみフレックス方向ユーティリティを適用するには、既存のクラス名に `{screen}:`プレフィックスを追加します。
たとえば、クラス`md:flex-row`を要素に追加すると、中程度の画面サイズ以上で`flex-row`ユーティリティが適用されます。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['flex-row', 'flex-col', 'flex-row-reverse', 'flex-col-reverse', 'flex-row']}
  snippet={(classNames) => `
    <div class="${classNames} ...">
      <!-- ... -->
    </div>
  `}
  preview={(className) => (
    <div className={`flex bg-gray-200 ${className}`}>
      <div className="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">1</div>
      <div className="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">2</div>
      <div className="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">3</div>
    </div>
  )}
/>
