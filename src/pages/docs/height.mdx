## Auto

`h-auto`を使用して、ブラウザに要素の高さを決定させます。

```html
<template preview>
  <div class="h-auto w-32 text-center mx-auto bg-gray-400 p-6">h-auto</div>
</template>

<div class="h-auto w-32 p-6 ...">h-auto</div>
```

## Screen height

`h-screen`を使用して、要素がビューポートの高さ全体に広がるようにします。

```html
<template preview class="overflow-x-scroll p-4">
  <div class="bg-gray-400 h-screen"></div>
</template>
```

## Fixed height

要素を固定の高さに設定するには、`h-{number}`または`h-px`を使用します。

```html
<template preview class="flex justify-around items-end text-sm p-4">
  <div class="mr-3">
    <div class="h-8 w-8 bg-gray-400"></div>
    <p class="text-center mt-2 text-sm">h-8</p>
  </div>
  <div class="mr-3">
    <div class="h-12 w-12 bg-gray-400"></div>
    <p class="text-center mt-2 text-sm">h-12</p>
  </div>
  <div>
    <div class="h-16 w-16 bg-gray-400"></div>
    <p class="text-center mt-2 text-sm">h-16</p>
  </div>
</template>

<div class="h-8 w-8 ..."></div>
<div class="h-12 w-12 ..."></div>
<div class="h-16 w-16 ..."></div>
```

## Full height

親が定義された高さを持っている限り`h-full`を使用して、要素の高さをその親の100％に設定します。

```html
<template preview class="flex justify-around items-end text-sm p-4">
  <div class="h-48">
    <div class="h-full p-6 bg-gray-400">h-full</div>
  </div>
</template>

<div class="h-48">
  <div class="h-full p-6 bg-gray-400">h-full</div>
</div>
```

## Responsive

特定のブレークポイントで要素の高さを制御するには、既存の幅ユーティリティに`{screen}:`プレフィックスを追加します。
たとえば、クラス`md:h-full`を要素に追加すると、中程度の画面サイズ以上で`h-full`ユーティリティが適用されます。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['h-8', 'h-12', 'h-16', 'h-20', 'h-24']}
  snippet={(classNames) => `
    <div class="${classNames} w-32 bg-gray-400"></div>
  `}
  preview={(className) => (
    <div className={`w-32 bg-gray-400 ${className}`} />
  )}
/>

## カスタマイズ

### Height Scale

デフォルトでは、Tailwindの高さスケールは[default spacing scale](/docs/customizing-spacing#default-spacing-scale)と、高さに固有のいくつかの追加値の組み合わせです。

`tailwind.config.js`ファイルの`theme.spacing`セクションで、パディング、マージン、幅、高さの間隔スケールを一度にカスタマイズできます。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      spacing: {
+       sm: '8px',
+       md: '16px',
+       lg: '24px',
+       xl: '48px',
      }
    }
  }
```

高さを個別にカスタマイズするには`tailwind.config.js`ファイルの`theme.height`セクションを使用します。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      height: {
+       sm: '8px',
+       md: '16px',
+       lg: '24px',
+       xl: '48px',
      }
    }
  }
```
