## Responsive

特定のブレークポイントで要素のclearプロパティを制御するには、既存のclearユーティリティに`{screen}:`プレフィックスを追加します。
たとえば、`md:clear-left`を使用して、中程度の画面サイズ以上でのみ`clear-left`ユーティリティを適用します。

```html
<p class="clear-right md:clear-left ...">
  <!-- ... -->
</p>
```

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。
