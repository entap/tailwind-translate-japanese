## Auto

`.w-auto`を使用して、ブラウザに要素の幅を計算させて選択させます。

```html
<template preview>
  <div class="w-auto bg-gray-400 mb-4 px-2">.w-auto on block element</div>
  <div class="w-auto inline-block bg-gray-400 px-2">.w-auto on inline-block element</div>
</template>

<div class="w-auto ..."></div>
<div class="w-auto inline-block ..."></div>
```

## Screen Width

`.w-screen`を使用して、要素をビューポートの幅全体に広げます。

```html
<template preview class="overflow-x-scroll p-4">
  <div class="w-screen bg-gray-400 h-4"></div>
</template>
```

## Fluid Width

`.w-{fraction}`または`.w-full`を使用して、要素をパーセンテージベースの幅に設定します。

```html

    <div class="flex mb-4">

      <div class="w-1/2 p-2 bg-gray-500 text-center">.w-1/2</div>
    </div>
    <div class="flex mb-4">
      <div class="w-2/5 p-2 bg-gray-400 text-center">.w-2/5</div>
      <div class="w-3/5 p-2 bg-gray-500 text-center">.w-3/5</div>
    </div>
    <div class="flex mb-4">
      <div class="w-1/3 p-2 bg-gray-400 text-center">.w-1/3</div>
      <div class="w-2/3 p-2 bg-gray-500 text-center">.w-2/3</div>
    </div>
    <div class="flex mb-4">
      <div class="w-1/4 p-2 bg-gray-400 text-center">.w-1/4</div>
      <div class="w-3/4 p-2 bg-gray-500 text-center">.w-3/4</div>
    </div>
    <div class="flex mb-4">
      <div class="w-1/5 p-2 bg-gray-400 text-center">.w-1/5</div>
      <div class="w-4/5 p-2 bg-gray-500 text-center">.w-4/5</div>
    </div>
    <div class="flex mb-4">
      <div class="w-1/6 p-2 bg-gray-400 text-center">.w-1/6</div>

    <div class="w-full p-2 bg-gray-400 text-center">.w-full</div>
  </div>
</template>
```

## Responsive

特定のブレークポイントで要素の幅を制御するには、既存の幅ユーティリティに`{screen}:`プレフィックスを追加します。
たとえば、クラス`md:w-full`を要素に追加すると、中程度の画面サイズ以上で`w-full`ユーティリティが適用されます。

Tailwindのレスポンシブデザイン機能の詳細については、[レスポンシブデザイン](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['w-1/2', 'w-auto', 'w-full', 'w-32', 'w-3/4']}
  snippet={(classNames) => `
    <div class="bg-gray-400 p-4 text-center">
      <div class="${classNames} ...">
        Responsive Element
      </div>
    </div>
  `}
  preview={(className) => (
    <div className="bg-gray-400 p-4 text-center">
      <div className={`inline-block bg-gray-800 text-white p-2 truncate ${className}`}>Responsive Element</div>
    </div>
  )}
/>

## Customizing

### Width scale

デフォルトでは、Tailwindの幅スケールは、[デフォルトの間隔スケール](/docs/customizing-spacing#default-spacing-scale)と、幅に固有のいくつかの追加値の組み合わせです。

`tailwind.config.js`ファイルの`theme.spacing`または`theme.extend.spacing`セクションで、パディング、マージン、幅、高さの間隔スケールを一度にカスタマイズできます。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
+         '72': '18rem',
+         '84': '21rem',
+         '96': '24rem',
        }
      }
    }
  }
```

幅を個別にカスタマイズするには、`tailwind.config.js`ファイルの`theme.width`セクションを使用します。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        width: {
+         '1/7': '14.2857143%',
+         '2/7': '28.5714286%',
+         '3/7': '42.8571429%',
+         '4/7': '57.1428571%',
+         '5/7': '71.4285714%',
+         '6/7': '85.7142857%',
        }
      }
    }
  }
```

デフォルトテーマのカスタマイズについては[テーマのカスタマイズ](/docs/theme#customizing-the-default-theme)をご覧ください。
