## Include borders and padding

`box-border`を使用して、要素の` box-sizing`を `border-box`に設定し、高さまたは幅を指定するときに要素の境界線とパディングを含めるようにブラウザに指示します。

これは、2pxの境界線とすべての側面に4pxのパディングがある100px &times; 100pxの要素が、88px &times; 88pxの内部コンテンツ領域で100px &times; 100pxとしてレンダリングされることを意味します。

Tailwindは、これを[preflight base styles](/docs/preflight)のすべての要素のデフォルトにします。

```html
<template preview class="bg-white p-8 flex items-center justify-around">
  <div class="box-border h-20 w-32 p-4 border-4 border-gray-400 bg-gray-200">
    <div class="h-full w-full bg-gray-400"></div>
  </div>
</template>

<div class="box-border h-20 w-32 p-4 border-4 border-gray-400 bg-gray-200">
  <div class="h-full w-full bg-gray-400"></div>
</div>
```

## Exclude borders and padding

`box-content`を使用して、要素の` box-sizing`を `content-box`に設定し、要素の指定された幅または高さに境界線とパディングを追加するようにブラウザに指示します。

つまり、2pxの境界線と4pxのパディングをすべての側面に持つ100px &times; 100pxの要素は、実際には112px &times; 112pxとしてレンダリングされ、内部コンテンツ領域は100px &times; 100pxになります。

```html
<template preview class="bg-white p-8 flex items-center justify-around">
  <div class="box-content h-20 w-32 p-4 border-4 border-gray-400 bg-gray-200">
    <div class="h-full w-full bg-gray-400"></div>
  </div>
</template>

<div class="box-content h-20 w-32 p-4 border-4 border-gray-400 bg-gray-200">
  <div class="h-full w-full bg-gray-400"></div>
</div>
```

## Responsive

特定のブレークポイントでボックスサイズを制御するには、既存のボックスサイズユーティリティに `{screen}:`プレフィックスを追加します。 たとえば、`md:box-content`を使用して、中程度の画面サイズ以上でのみ`box-content`ユーティリティを適用します。

```html
<div class="box-border md:box-content ...">
  <!-- ... -->
</div>
```

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。
