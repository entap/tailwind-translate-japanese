## 使用方法

`max-h-full`または`max-h-screen`ユーティリティを使用して要素の最大高さを設定します。

```html
<template preview>
  <div class="h-24 p-6 bg-gray-300">
    <div class="h-48 max-h-full p-6 bg-gray-400 flex items-center justify-center">
      <span>max-h-full</span>
    </div>
  </div>
</template>

<div class="h-24 ...">
  <div class="h-48 max-h-full ...">
    max-h-full
  </div>
</div>
```

## レスポンシブ

特定のブレークポイントで要素の最大高さを制御するには、既存の最大高さユーティリティに`{screen}:`プレフィックスを追加します。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['max-h-full', 'max-h-screen', 'max-h-full', 'max-h-screen', 'max-h-full']}
  snippet={(classNames) => `
    <div class="h-24 ...">
      <div class="h-48 ${classNames} ...">
        <span>Target</span>
      </div>
    </div>
  `}
  preview={(className) => (
    <div className="h-24 p-6 bg-gray-300">
      <div className={`h-48 p-6 bg-gray-400 flex items-center justify-center ${className}`}>
        <span>Target</span>
      </div>
    </div>
  )}
/>

## カスタマイズ

### Max-height scale

`tailwind.config.js`ファイルの`theme.maxHeight`セクションでTailwindのデフォルトの最大高さスケールをカスタマイズします。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      maxHeight: {
+       '0': '0',
+       '1/4': '25%',
+       '1/2': '50%',
+       '3/4': '75%',
+       'full': '100%',
      }
    }
  }
```

デフォルトテーマのカスタマイズについては[theme customization documentation](/docs/theme#customizing-the-default-theme)をご覧ください。
