## レスポンシブ

特定のブレークポイントで要素のフォントサイズを制御するには、既存のフォントサイズユーティリティに `{screen}:`プレフィックスを追加します。
たとえば、`md:text-lg`を使用して、中程度の画面サイズ以上でのみ`text-lg`ユーティリティを適用します。

Tailwindのレスポンシブデザイン機能の詳細については、[Responsive Design](/docs/responsive-design)のドキュメントをご覧ください。

<ResponsiveCodeSample
  classNames={['text-base', 'text-lg', 'text-xl', 'text-2xl', 'text-3xl']}
  snippet={(classNames) => `
    <p class="${classNames} ...">The quick brown fox jumped over the lazy dog.</p>
  `}
  preview={(className) => (
    <p className={`text-gray-800 truncate ${className}`}>The quick brown fox jumped over the lazy dog.</p>
  )}
/>

## カスタマイズ

### フォントサイズ

デフォルトでは、Tailwindは10個の`font-size`ユーティリティを提供します。
Tailwindコンフィグの`theme.fontSize`セクションを編集して、これらを変更、追加、または削除します。

```diff
  // tailwind.config.js
  module.exports = {
    theme: {
      fontSize: {
-       'xs': '.75rem',
-       'sm': '.875rem',
+       'tiny': '.875rem',
        'base': '1rem',
        'lg': '1.125rem',
        'xl': '1.25rem',
        '2xl': '1.5rem',
-       '3xl': '1.875rem',
-       '4xl': '2.25rem',
        '5xl': '3rem',
        '6xl': '4rem',
+       '7xl': '5rem',
      }
    }
  }
```

### <Heading badge="v1.3.0+">デフォルトの行の高さを提供する</Heading>

`tailwind.config.js`ファイルの`[fontSize、lineHeight]`形式のタプルを使用して、各フォントサイズのデフォルトの行の高さを指定できます。

```js
// tailwind.config.js
module.exports = {
  theme: {
    fontSize: {
      sm: ['14px', '20px'],
      base: ['16px', '24px'],
      lg: ['20px', '28px'],
      xl: ['24px', '32px'],
    }
  }
}
```

下位互換性の理由から、設定せずに実行することはありませんが、独自のプロジェクトで作成するのは非常に便利なカスタマイズになる可能性があります。

### <Heading badge="v1.7.0+">デフォルトの文字間隔を提供する</Heading>

フォントサイズのデフォルトの文字間隔値も指定する場合は、`tailwind.config.js`ファイルで`[fontSize、{letterSpacing、lineHeight}]`の形式のタプルを使用して指定できます。

```js
// tailwind.config.js
module.exports = {
  theme: {
    fontSize: {
      '2xl': ['24px', {
        letterSpacing: '-0.01em',
      }],
      // Or with a default line-height as well
      '3xl': ['32px', {
        letterSpacing: '-0.02em',
        lineHeight: '40px',
      }],
    }
  }
}
```
