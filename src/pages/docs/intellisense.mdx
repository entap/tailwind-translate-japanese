[Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)は、オートコンプリート、構文の強調表示、リンティングなどの高度な機能をVisual Studio Codeユーザーに提供することにより、Tailwind開発エクスペリエンスを強化します。
<img src={require('@/img/intellisense.png').default} alt="" width={1344} height={672} />

- **Autocomplete**. クラス名や[CSS functions and directives](/docs/functions-and-directives)のインテリジェントな提案。
- **Linting**. CSSとマークアップの両方のエラーと潜在的なバグを強調表示します。
- **Hover Previews**. Tailwindクラス名にカーソルを合わせると、完全なCSSが表示されます。
- **Syntax Highlighting**. Tailwind機能が正しく強調表示されるように構文定義を提供します。

より詳しくは[GitHub](https://github.com/tailwindcss/intellisense)のプロジェクトを参照したり、今すぐ始めるために[Visual Studio Code](vscode:extension/bradlc.vscode-tailwindcss)に追加してください。
